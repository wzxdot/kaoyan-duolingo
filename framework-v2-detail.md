## 二、英语一模块

### 2.1 真题学习节奏

从 2026 年真题开始，倒序推进至 1998 年，每份真题用 14 天完成，具体日程如下：

| 天数 | 学习内容 | 是否做真题 | 复习重点 |
| --- | --- | --- | --- |
| 第 1 天 | Reading Text 1：单词、词组、长难句 + 真题训练 | 是 | 错题与不熟悉的语言点 |
| 第 2 天 | 复习第 1 天单词、词组、句子；不做新真题 | 否 | 标记为生词的词汇、模糊词组 |
| 第 3 天 | Reading Text 2：单词、词组、长难句 + 真题训练 | 是 | 同 Text 1 |
| 第 4 天 | 复习 Text 2 及相关语言点 | 否 | 生词与错误句子 |
| 第 5 天 | Reading Text 3 | 是 | — |
| 第 6 天 | 复习 Text 3 | 否 | — |
| 第 7 天 | Reading Text 4 | 是 | — |
| 第 8 天 | 复习 Text 4 | 否 | — |
| 第 9 天 | 翻译：真题训练 + 逐句拆解 | 是 | 翻译技巧与句式 |
| 第 10 天 | 复习翻译中的单词、词组、句式 | 否 | — |
| 第 11 天 | 完形填空：真题训练 | 是 | 上下文逻辑与固定搭配 |
| 第 12 天 | 复习完形中的单词、词组、句子 | 否 | — |
| 第 13 天 | 新题型：真题训练 | 是 | 段落匹配、排序等技巧 |
| 第 14 天 | 复习新题型中的单词、词组、句子；本周总复习 | 否 | — |

翻译、完型、新题型各环节均融入「考研单词 + 词组 + 句子」学习，不只是答题技巧训练。

2026 年至 1998 年共 29 份真题，按此节奏约需 58 周（约 13.5 个月）。当前考研倒计时约 161 天，若目标为 2026 年 12 月考试，该计划会超出考试日期。建议确认：是否用于 2027 年备考，或仅选取最近 10-15 年真题在考前完成。

### 2.2 每日学习单元设计

每个 Text / 题型作为一个学习单元，单元内包含：

1. 单词预习：从真题文章中提取的核心词汇，附带音标、释义、例句。
2. 词组学习：真题中出现的固定搭配与高频短语。
3. 长难句拆解：逐句分析语法结构，支持点击查看翻译。
4. 真题训练：对应 Text 或题型的 4-5 道题目。
5. 错题/生词归因：答错的题目自动进入错题本，不熟悉的单词加入生词本。

### 2.3 生词本（游戏闯关模式）

生词本仅服务于英语一，包含三类内容：单词、词组、句子。整体采用 Duolingo 式闯关路径：

- 按来源真题年份与 Text 分组，每组为一个「单词关 / 词组关 / 句子关」。
- 每关 10-15 个条目，完成当前关后解锁下一关。
- 单词卡片正面为单词与音标，背面为释义、例句、真题原句。
- 词组卡片正面为短语，背面为释义与真题例句。
- 句子卡片正面为长难句，背面为结构分析与译文。
- 用户选择「认识 / 模糊 / 不认识」：
  - 认识：间隔翻倍，下次复习时间延后。
  - 模糊：间隔不变，次日继续复习。
  - 不认识：重置间隔，次日复习。
- 连续答对获得连胜、XP、宝石奖励。
- 每日推送「今日待复习关」，按遗忘曲线排序。

### 2.4 交互功能保留

- 点击查看翻译：文章、题目、选项中任意句子支持点击展开译文。
- 长按单词翻译：长按文章或题目中的单词，弹出释义与「加入生词本」按钮。

## 三、数学二模块

### 3.1 章节与学习路径

以张宇考研数学二 2026 版章节为骨架，按以下顺序学习，知识点可继续细分：

| 章节 | 建议学习周期 | 难度 |
| --- | --- | --- |
| 函数极限与连续 | 1 周 | 中 |
| 数列极限 | 0.5 周 | 中 |
| 一元函数微分学 | 2 周 | 高 |
| 一元函数积分学 | 2 周 | 高 |
| 多元函数微分学 | 1.5 周 | 中高 |
| 二重积分 | 1 周 | 中 |
| 常微分方程 | 1 周 | 中 |
| 行列式 | 0.5 周 | 低 |
| 矩阵 | 1 周 | 中 |
| 向量组与线性方程组 | 1.5 周 | 中高 |
| 特征值与特征向量 | 1 周 | 中 |
| 二次型 | 1 周 | 中 |

总计约 13.5 周，加上复习与弹性时间，6 个月内可完成。每个知识点节点包含：

1. 知识点讲解：概念、定理、公式。
2. 细分例题：1-3 道例题，支持分步查看解析。
3. 张宇 1000 题训练：从 2026 版中抽取对应知识点题目。
4. 错题回顾：第二天自动推送前一天错误题目。

### 3.2 学习节奏

- 第 1 天：学习新知识点 + 完成例题与 1000 题训练。
- 第 2 天：复习昨日错题，补充同类变式题。
- 难度高、知识点多的章节可延长至 3 天。
