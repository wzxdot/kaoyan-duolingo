# 考研闯关 V2 升级框架

## 升级目标

在现有 Duolingo 式闯关原型基础上，参考扇贝考研的核心学习闭环，增加：

1. **生词本功能**：学习过程中随时收藏生词，支持背诵、复习与遗忘曲线提醒。
2. **历年真题模块**：按科目/年份/题型组织真题，支持练习模式与模拟考试模式。

整体保持简洁、轻量的前端原型定位，数据仍通过 `localStorage` 本地持久化。

---

## 一、生词本功能

### 1.1 核心能力

- **查词与收藏**：在答题、预习、复习过程中，遇到不认识的单词可一键加入生词本。
- **单词卡片**：正面显示单词/短语，背面显示释义、例句、音标、真题例句。
- **背诵模式**：基于艾宾浩斯遗忘曲线，每日推送需复习的单词。
- **熟词标记**：标记为“已掌握”的单词不再反复出现，但仍可查询。
- **导入导出**：支持从本地导入生词列表，或导出为 CSV/JSON 备份。

### 1.2 数据结构

```json
{
  "word": "ambiguous",
  "phonetic": "/æmˈbɪɡjuəs/",
  "meaning": "adj. 模棱两可的，含糊不清的",
  "example": "The meaning of the sentence remains ambiguous.",
  "tags": ["态度词", "阅读高频"],
  "status": "learning",
  "addedAt": "2026-07-18",
  "reviewAt": "2026-07-19",
  "reviewCount": 0,
  "source": "eng-1"
}
```

### 1.3 页面规划

| 页面/组件 | 说明 |
| --- | --- |
| 底部导航新增「单词」入口 | 与「学习」「复习」「排行」「我的」并列 |
| 单词本首页 | 今日待复习数量、已掌握数量、学习中数量、快捷开始复习按钮 |
| 单词卡片页 | 左右滑动翻牌、认识/模糊/不认识三级反馈 |
| 单词详情弹窗 | 在答题时长按/点击单词弹出释义与收藏按钮 |
| 我的-导入导出 | 支持批量导入生词与导出备份 |

### 1.4 复习算法（简化版 SM-2）

- 首次加入：次日复习。
- 认识：复习间隔 × 2（1天 → 2天 → 4天 → 8天…）。
- 模糊：间隔不变，次日再复习。
- 不认识：重置为次日复习。

## 二、历年真题模块

### 2.1 核心能力

- **按维度筛选**：年份（如 2010-2025）、科目（英语一/数学二/政治/材料）、题型（选择/填空/解答/翻译/作文）。
- **练习模式**：逐题作答，实时判分，查看解析，错题自动归入错题本。
- **模拟考试模式**：限时答题，提交后给出总分、各题型得分与用时分析。
- **收藏与笔记**：对真题添加收藏、笔记，便于二轮复习。

### 2.2 数据结构

```json
{
  "id": "eng-2024-01",
  "subject": "english",
  "year": 2024,
  "type": "reading",
  "title": "2024年英语一 阅读理解 Text 1",
  "passage": "...",
  "questions": [
    {
      "id": "q1",
      "type": "multiple",
      "question": "...",
      "options": ["...", "...", "...", "..."],
      "answer": 1,
      "explanation": "...",
      "difficulty": 3
    }
  ],
  "timeLimit": 1200,
  "source": "2024年考研英语一真题"
}
```

### 2.3 页面规划

| 页面/组件 | 说明 |
| --- | --- |
| 学习页新增「真题」Tab | 与科目路径并列，入口为「历年真题」 |
| 真题首页 | 展示最近真题、按年份/科目/题型筛选 |
| 真题练习页 | 单题练习，支持标记、笔记、收藏 |
| 真题模考页 | 整卷限时，顶部显示倒计时与答题卡 |
| 模考结果页 | 得分、正确率、用时、薄弱知识点分析 |

### 2.4 真题内容规划（首期）

| 科目 | 首批覆盖年份 | 题型 |
| --- | --- | --- |
| 考研英语一 | 2020-2024 | 完形、阅读、翻译、新题型 |
| 考研数学二 | 2020-2024 | 选择、填空、解答 |
| 考研政治 | 2020-2024 | 单选、多选、分析题 |
| 材料科学基础 | 精选 5 套 | 名词解释、简答、计算、综合分析 |

## 三、现有页面调整

### 3.1 底部导航

新增「单词」入口，顺序建议为：学习、单词、真题、复习、我的。

### 3.2 学习页

在科目路径上方增加「历年真题」快捷入口卡片。

### 3.3 答题页

- 长按/点击英文单词可弹出「查词/收藏」按钮。
- 数学题支持查看公式提示（可选）。

### 3.4 错题本

真题错题与闯关错题合并展示，支持按来源筛选。

## 六、待确认问题

1. **生词本范围**：是否只做英语一生词，还是四科都支持（如政治术语、材料专业名词）？
2. **真题来源**：是否由你提供历年真题文件/图片，还是我先基于公开真题整理成题库？
3. **界面风格**：单词页倾向扇贝式的列表+卡片，还是 Duolingo 式的游戏化闯关？
4. **数据迁移**：是否需要把当前 `localStorage` 数据导出为 JSON，方便后续同步到后端？

确认以上方向后，即可按阶段开始升级实现。
